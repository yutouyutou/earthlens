# EarthLens 🌍

See the world live — browse real-time cameras from around the globe on a 3D map.

## Tech Stack
- Next.js 14 + TypeScript
- Mapbox GL JS 3D Globe
- HLS.js / YouTube live streams
- Prisma + SQLite
- NextAuth (GitHub OAuth)
- Tailwind CSS

## Getting Started

```bash
npm install
npx prisma db push
npx prisma db seed
npm run dev
```

Open http://localhost:3000
